# 🏥 Medication Error Prediction System

Sistem prediksi medication error berbasis AI untuk penelitian tesis S2 Farmasi Klinis.

## 🚀 Live Demo

Visit: [https://medication-error-predictor.streamlit.app](https://medication-error-predictor.streamlit.app)

(Ganti dengan URL Anda setelah deploy)

## 📊 Fitur

- Prediksi risk score medication error secara real-time
- Dashboard interaktif dengan Streamlit
- Model Machine Learning: Random Forest
- 5 fitur prediktor: usia, jumlah obat, lama rawat, komorbiditas, beban kerja

## 🛠️ Tech Stack

- Python 3.10+
- Streamlit
- scikit-learn
- Pandas
- Plotly

## 💻 Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📝 Disclaimer

Prototype untuk keperluan penelitian. Data yang digunakan adalah simulasi.
Tidak untuk digunakan dalam praktik klinis sebenarnya.
